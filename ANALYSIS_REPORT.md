## 3. 분석 방법론 상세

### 3-1. Stage 1: 데이터 정규화

#### Sff (Supply Float Factor)
```
Sff = (순매수 금액 / 유통시총) × 100
```

**목적**: 시가총액 왜곡 제거
- 삼성전자 10억 매수 vs 중소형주 10억 매수 → 유통물량 대비 비율로 공정 비교

#### Z-Score
```
Z-Score = (현재값 - 60일 평균) / 60일 표준편차
```

**목적**: 변동성 보정
- Z > +2.0: 상위 2.5% 수준, 통계적 이상 수급 (강한 매수세)
- -2.0 < Z < 2.0: 중립
- Z < -2.0: 하위 2.5% 수준, 통계적 이상 수급 (강한 매도세)

### 3-2. Stage 2: 시공간 분석

#### 6개 기간 Z-Score 계산

| 기간 | 거래일 수 | 의미 |
|------|-----------|------|
| 1W | 5일 | 단기 수급 강도 |
| 1M | 21일 | 월간 트렌드 |
| 3M | 63일 | 분기 트렌드 |
| 6M | 126일 | 반기 트렌드 |
| 1Y | 252일 | 연간 트렌드 |
| 2Y | 504일 | 장기 트렌드 |

#### 4가지 정렬 키 계산

```python
Recent    = (1W + 1M) / 2                          # 현재 강도
Momentum  = 1W - 2Y                                # 수급 개선도 (전환점 포착)
Weighted  = (1W×6 + 1M×5 + 3M×4 + 6M×3 + 1Y×2 + 2Y×1) / 21  # 중장기 가중 트렌드
Average   = (1W + 1M + 3M + 6M + 1Y + 2Y) / 6      # 전체 일관성
```

### 3-3. Stage 3: 패턴 분류

#### 분류 로직

**1. 모멘텀형**
- 조건: `Momentum > 1.0 AND Recent > 1.5`
- 의미: 과거 약세 → 최근 강세 전환
- 전략: 단기 추격 매수
- 손절: -5%

**2. 지속형**
- 조건: `Weighted > 1.5 AND Average > 1.0`
- 의미: 중장기 일관된 매집 패턴
- 전략: 조정 시 분할 매수 (5~10% 조정)
- 손절: -10%

**3. 전환형**
- 조건: `Weighted > 0.5 AND Momentum < 0`
- 의미: 장기 강세 + 최근 약화 (고점에서 조정 중)
- 현재 상태: 고점에서 조정 진행 중
- 전략: 저점 매수 대기 (반등 시그널 확인 후 진입)
- 손절: -7%
- 참고: 전환형의 95% 이상이 2Y > 0 (장기 강세 상태)

**4. 기타**
- 조건: 위 조건 모두 미충족
- 의미: 명확한 패턴 없음
- 전략: 관망

#### 점수 계산 공식

```python
# 패턴별 핵심 정렬 키 선택
if pattern == '모멘텀형':
    key_score = momentum
elif pattern == '지속형':
    key_score = weighted
elif pattern == '전환형':
    key_score = recent
else:
    key_score = average

# 점수 계산 (0~100점 범위)
pattern_strength = (key_score - 0.5) × 20
final_score = min(100, max(0, pattern_strength))
```

### 3-4. 패턴 vs 시그널: 수식 정의 및 비교 분석

#### 3-4-1. 정렬 키 상세 수식 (소스 코드 기반)

**실제 구현된 수식** (`src/analyzer/pattern_classifier.py`):

```python
# 기초 변수: 6개 기간 Z-Score
1W, 1M, 3M, 6M, 1Y, 2Y

# 정렬 키 계산
Recent = (1W + 1M) / 2

Momentum = 1W - 2Y

Weighted = (1W×3.0 + 1M×2.5 + 3M×2.0 + 6M×1.5 + 1Y×1.0 + 2Y×0.5) / 10.5

Average = (1W + 1M + 3M + 6M + 1Y + 2Y) / 6

Persistence = (양수 Z-Score 기간 개수) / 6
```

#### 3-4-2. 패턴 분류 상세 규칙

**실제 임계값** (소스 코드):

| 패턴 | 조건 1 | 조건 2 | 의미 |
|------|--------|--------|------|
| 모멘텀형 | `Momentum > 1.0` | `Recent > 0.5` | 1W - 2Y > 1.0 (급등) |
| 지속형 | `Weighted > 0.8` | `Persistence > 0.7` | 가중평균 > 0.8 + 83% 이상 양수 |
| 전환형 | `Weighted > 0.5` | `Momentum < 0` | 1W < 2Y (조정 중) |

**점수 계산 상세 공식**:
```python
Weighted_sum = Recent×0.25 + Momentum×0.25 + Weighted×0.30 + Average×0.20
Score = ((Weighted_sum + 3) / 6) × 100
Final_score = Clip(Score, 0, 100)
```

**변환 예시**:
- Weighted_sum = +3.0 → 100점 (최고)
- Weighted_sum = 0.0 → 50점 (중립)
- Weighted_sum = -3.0 → 0점 (최저)

#### 3-4-3. 시그널 탐지 상세 수식

**[시그널 1] MA 골든크로스** (`src/analyzer/signal_detector.py`):
```python
MA_short(5일) = 최근 5일 외국인 순매수 이동평균
MA_long(20일) = 최근 20일 외국인 순매수 이동평균

조건:
  현재일: MA_short > MA_long
  직전일: MA_short ≤ MA_long

의미: 단기 수급이 중기 수급을 상향 돌파
```

**[시그널 2] 수급 가속도**:
```python
Recent_avg = 최근 5일 (외국인+기관) 순매수 평균
Prev_avg = 직전 5일 (외국인+기관) 순매수 평균
Acceleration = Recent_avg / Prev_avg

조건: Acceleration > 1.5

의미: 수급이 1.5배 이상 가속 (매수세 강화)
```

**[시그널 3] 외인-기관 동조율**:
```python
Sync_days = (외인 순매수 > 0 AND 기관 순매수 > 0) 인 일수
Sync_rate = (Sync_days / 20일) × 100%

조건: Sync_rate > 75%

의미: 외인과 기관이 75% 이상 동시 매수 (확신도 높음)
```
